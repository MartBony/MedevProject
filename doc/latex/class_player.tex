\doxysubsection{Detailed Description}
CLasse représentant un joueur dans le jeu. 

Cette classe permet d\textquotesingle{}initier un joueur et ses grilles, puis de jouer à travers le terminal. Elle ne peut pas exister seule et nécessite un autre joueur omme adversaire.
